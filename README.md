

## 简介

基于Nuxt开发的网盘资源搜索引擎



## 安装指南

1. 克隆项目到本地：

```bash
git clone https://github.com/Amd794/nuxt_cloud_seeker.git
```

2. 进入项目目录：

```bash
cd nuxt_cloud_seeker
```

3. 安装依赖：

```bash
npm install
```

4. 启动开发服务器：

```bash
npm run dev
```
